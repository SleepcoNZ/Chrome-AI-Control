Task Stacking (Plans)

Configure how Aria handles multi-step tasks. When you chain actions together (e.g. "open Trade Me, search for kettlebells, find the cheapest"), Aria creates a plan and executes each step.

When enabled, Aria will immediately start running the plan after creating it. When disabled, you'll see the plan and choose to run all steps or step through manually.

Pauses before actions like form submissions, purchases, or account changes so you can confirm or cancel.

Show progress messages in chat as each step runs. Turn off for a cleaner chat with just the final result.

Show a browser notification when a multi-step plan finishes, useful if you've switched to another tab.
